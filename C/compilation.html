Compilation process of C language

when we write a c code we use a .c extension and the file contains english like letters.We call it as source code, but we know that Computer understands the 0 and 1 only. So to convert the code to Higher level language(English like ) to lower level (00 and 11 ) is called compilation process and it is done by mainly 4 components

  1. Preprocessor
  2. Compilar
  3. Assembler
  4. Linker

Preprocessors

Our source code has many unnecessary lines as comments,preprocessor removes them and all those line including #include is expanded by including the respective liberary

Compiler

The code which is expanded by the preprocessor is compiled by compilar and convert it into assembly language.

Assembler

Assembler takes the assembly code and convert it into the object code and create a file having same name as source file but has extension .obj

Linker

Linker links the library file's object code with the source code. In DOS, the extension of the executable file is '.exe', and in UNIX, the executable file can be named as 'a.out'. The output of liker is an executable file having extension .exe
