Introduction to C Language

C Language is one on the popular programming language developed by Dennis Ritchie. It wa build to write Operating systems It is consider as a base for other languages. Many language syntax liks Java PHP Javascript are based on C language. C++ is an extension of C.
There are 3 level of languages

  1. Low level language
  2. mid level language
  3. High level language

C is a mid level language because it has the property of both high level lngauges and low level languages

Features of C language

  1. Easy
  2. Fast
  3. Popular
  4. Structured
  5. pointer
  6. Recursion

C is Very easy to learn because its syntax is easy and short. It is fast because there is less building functions. While it was a very fundamenta programming language thats why it is still popular world wide and taught as the first programming lagnuage in many universities

C is a structured language which means it contains functions to do some specific tasks.

Pointer is a way to deal with memory drectly. (You will learn about it later)
