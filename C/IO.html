Input Output

Those methods by which we can take input from the user and give output on the console is called Input / output functions

Printf();

As you remember from your first code, Printf is use to print on the console. It can be use to print a string (Stream of line ) or numbers. Let us see it as an example

#include <stdio.h>

int main(){

printf("This site is best to learn");

}

Output: This site is best to learn

scanf();

Scanf is use to take the input from the users, but before taking input you have to specify which type of input will you are going to take and where will you store it

scanf("type of input", memory addres where u store);

example :- scanf("%d",&a);

#include <stdio.h>

int main(){

int a; //creating an integer container to store number

scanf("%d",&a); //taking input

printf("%d",a); //printing output

}

Input: 35
Output:35

We can take multiple inputs at one time
format of scanf: scanf("%d%d",&a,&b);
it will take 2 input integer and store it in a and b

Make a calculator

we will make a calculator where we will get 2 input number and we will add subtract multiply and divide them

#include <stdio.h>

int main(){

int a,b; //creating an integer container to store number

scanf("%d%d",&a,&b); //taking input

printf(" addition of a and b is %d \n",a+b); //printing a+b

printf(" subtraction of a and b is %d \n",a-b); //printing a-b

printf(" multiplication of a and b is %d \n",a*b); //printing a*b

printf(" divison of a and b is %d \n",a/b); //printing a/b

}
